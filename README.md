# Data_Visualization_Projects

### 1. Analiza Skuteczności Strzałów w Piłce Nożnej (2017-2025)
Projekt analityczny badający efektywność zawodników w oparciu o metryki, takie jak **xG (Oczekiwane gole)** czy ilość oddanych strzałów.

* **Główne cele:** * Analiza porównawcza skuteczności strzałów w 5 najlepszych ligach świata (Premier League, La Liga, Serie A, Bundesliga, Ligue 1).
    * Badanie trendów na przestrzeni 8 sezonów (2017/18 - 2024/25).
    * Identyfikacja zawodników o najwyższym współczynniku *over-performing* względem modelu xG.
* **Technologie:** **R**
