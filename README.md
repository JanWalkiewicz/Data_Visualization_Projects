# Data_Visualization_Projects

### 1. Analiza_skuteczności_strzałów_w_piłce_nożnej - projekt oparty na języku programowania R
**Celem** projektu jest analiza różnic w skuteczności strzałów między zawodnikami w 5 najlepszych ligach świata (Premier League, La Liga, Serie A, Bundesliga oraz Ligue 1)  w sezonach 2017/18 - 2024/25.
