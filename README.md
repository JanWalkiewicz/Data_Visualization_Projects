# Data_Visualization_Projects

## PL

### 1. [Analiza_skuteczności_strzałów_w_piłce_nożnej](./Analiza_skuteczności_strzałów_w_piłce_nożnej) - projekt oparty na języku programowania R
**Celem** projektu jest wizualizacja, analiza oraz próba wyciągnięcia wniosków na podstawie rzeczywistych danych meczowy dotyczących strzałów w piłce nożnej w 5 najlepszych ligach świata (Premier League, La Liga, Serie A, Bundesliga oraz Ligue 1) w sezonach 2017/18 - 2024/25. Dane zostały zaczerpnięte z platformy [fbref](https://fbref.com/en/)
